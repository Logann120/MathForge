# Quadratic equations by factoring Worksheet

## Instructions

Solve each quadratic equation by factoring.

## Problems

1. Solve by factoring: x\*\*2 - 2\*x - 3 = 0
2. Solve by factoring: x\*\*2 - 2\*x - 8 = 0
3. Solve by factoring: x\*\*2 - 2\*x - 15 = 0

## Solution Key

1. Answer: -1, 3

   - Start with x\*\*2 - 2\*x - 3 = 0.
   - Factor the quadratic as (x + 1)(x - 3) = 0.
   - Set each factor equal to zero.
   - x = -1 or x = 3.
2. Answer: -2, 4

   - Start with x\*\*2 - 2\*x - 8 = 0.
   - Factor the quadratic as (x + 2)(x - 4) = 0.
   - Set each factor equal to zero.
   - x = -2 or x = 4.
3. Answer: -3, 5

   - Start with x\*\*2 - 2\*x - 15 = 0.
   - Factor the quadratic as (x + 3)(x - 5) = 0.
   - Set each factor equal to zero.
   - x = -3 or x = 5.
