# Factoring techniques Worksheet

## Instructions

Factor each polynomial expression completely.

## Problems

1. Factor completely: 6\*x\*\*2 + 8\*x
2. Factor completely: 4\*x\*\*2 - 9
3. Factor completely: x\*\*2 + x - 6

## Solution Key

1. Answer: 2\*x\*(3\*x + 4)

   - Start with 6\*x\*\*2 + 8\*x.
   - Identify the strategy: greatest common factor with variable.
   - Rewrite the expression as 2\*x\*(3\*x + 4).
   - Check by expanding the factored form.
2. Answer: (2\*x - 3)\*(2\*x + 3)

   - Start with 4\*x\*\*2 - 9.
   - Identify the strategy: coefficient difference of squares.
   - Rewrite the expression as (2\*x - 3)\*(2\*x + 3).
   - Check by expanding the factored form.
3. Answer: (x - 2)\*(x + 3)

   - Start with x\*\*2 + x - 6.
   - Identify the strategy: mixed-sign simple trinomial.
   - Rewrite the expression as (x - 2)\*(x + 3).
   - Check by expanding the factored form.
