# Systems of linear equations Worksheet

## Instructions

Solve each system of equations for x and y.

## Problems

1. Solve the system of equations:
2\*x + y = 7
x - 2\*y = -4
2. Solve the system of equations:
2\*x + y = 10
x - 2\*y = -5
3. Solve the system of equations:
2\*x + y = 13
x - 2\*y = -6

## Solution Key

1. Answer: (2, 3)

   - Start with the system 2\*x + y = 7 and x - 2\*y = -4.
   - Multiply the first equation by 2 so the y-coefficients are opposites.
   - Add the scaled first equation to the second equation to eliminate y.
   - Solve for x to get x = 2.
   - Substitute x = 2 into one equation to get y = 3.
   - The solution is (2, 3).
2. Answer: (3, 4)

   - Start with the system 2\*x + y = 10 and x - 2\*y = -5.
   - Multiply the first equation by 2 so the y-coefficients are opposites.
   - Add the scaled first equation to the second equation to eliminate y.
   - Solve for x to get x = 3.
   - Substitute x = 3 into one equation to get y = 4.
   - The solution is (3, 4).
3. Answer: (4, 5)

   - Add the scaled first equation to the second equation to eliminate y.
