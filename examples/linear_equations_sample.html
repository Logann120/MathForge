Linear equations Worksheet

Instructions

Solve each equation for x.

Problems

  1. Solve for x: 2*x + 1 = 3
  2. Solve for x: 3*x + 3 = 9
  3. Solve for x: 4*x + 5 = 17
  4. Solve for x: 5*x + 7 = 27
  5. Solve for x: 6*x + 9 = 39

Solution Key

  1. Answer: 1

    1. Start with 2*x + 1 = 3.
    2. Subtract 1 from both sides.
    3. Divide both sides by 2.
    4. x = 1.

  2. Answer: 2

    1. Start with 3*x + 3 = 9.
    2. Subtract 3 from both sides.
    3. Divide both sides by 3.
    4. x = 2.

  3. Answer: 3

    1. Start with 4*x + 5 = 17.
    2. Subtract 5 from both sides.
    3. Divide both sides by 4.
    4. x = 3.

  4. Answer: 4

    1. Start with 5*x + 7 = 27.
    2. Subtract 7 from both sides.
    3. Divide both sides by 5.
    4. x = 4.

  5. Answer: 5

    1. Start with 6*x + 9 = 39.
    2. Subtract 9 from both sides.
    3. Divide both sides by 6.
    4. x = 5.
