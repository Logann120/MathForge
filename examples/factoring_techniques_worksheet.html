Factoring techniques Worksheet

Instructions

Factor each polynomial expression completely.

Problems

  1. Factor completely: 6*x + 8
  2. Factor completely: x**2 - 4
  3. Factor completely: x**2 + 5*x + 6

Solution Key

  1. Answer: 2*(3*x + 4)

    1. Start with 6*x + 8.
    2. Identify the strategy: greatest common factor.
    3. Rewrite the expression as 2*(3*x + 4).
    4. Check by expanding the factored form.

  2. Answer: (x - 2)*(x + 2)

    1. Start with x**2 - 4.
    2. Identify the strategy: difference of squares.
    3. Rewrite the expression as (x - 2)*(x + 2).
    4. Check by expanding the factored form.

  3. Answer: (x + 2)*(x + 3)

    1. Start with x**2 + 5*x + 6.
    2. Identify the strategy: simple trinomial.
    3. Rewrite the expression as (x + 2)*(x + 3).
    4. Check by expanding the factored form.
