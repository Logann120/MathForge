# Quadratic equations by factoring Worksheet

## Instructions

Solve each quadratic equation by factoring.

## Problems

1. Solve by factoring: 2\*x\*\*2 - 8\*x + 6 = 0
2. Solve by factoring: 3\*x\*\*2 - 18\*x + 24 = 0
3. Solve by factoring: 4\*x\*\*2 - 32\*x + 60 = 0

## Solution Key

1. Answer: 1, 3

   - Start with 2\*x\*\*2 - 8\*x + 6 = 0.
   - Factor the quadratic as (2\*x - 2)(x - 3) = 0.
   - Set each factor equal to zero.
   - x = 1 or x = 3.
2. Answer: 2, 4

   - Start with 3\*x\*\*2 - 18\*x + 24 = 0.
   - Factor the quadratic as (3\*x - 6)(x - 4) = 0.
   - Set each factor equal to zero.
   - x = 2 or x = 4.
3. Answer: 3, 5

   - Start with 4\*x\*\*2 - 32\*x + 60 = 0.
   - Factor the quadratic as (4\*x - 12)(x - 5) = 0.
   - Set each factor equal to zero.
   - x = 3 or x = 5.
