# Functions basics Worksheet

## Instructions

Evaluate and interpret each function notation problem.

## Problems

1. Given f(x) = x\*\*2 + x + 2, evaluate f(2).
2. In the statement h(3) = 10, what ordered pair is represented?
3. Determine the domain of f(x) = sqrt(x - 2).

## Solution Key

1. Answer: 8

   - Start with f(x) = x\*\*2 + x + 2.
   - Substitute x = 2.
   - Evaluate to get f(2) = 8.
2. Answer: (3, 10)

   - Read h(3) as the output when the input is 3.
   - The input is 3 and the output is 10.
   - The ordered pair is (3, 10).
3. Answer: All real numbers x \>= 2

   - Start with f(x) = sqrt(x - 2).
   - The expression inside the square root must be nonnegative.
   - Solve x - 2 \>= 0 to get x \>= 2.
   - The domain is All real numbers x \>= 2.
