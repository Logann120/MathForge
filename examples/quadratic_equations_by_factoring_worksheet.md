# Quadratic equations by factoring Worksheet

## Instructions

Solve each quadratic equation by factoring.

## Problems

1. Solve by factoring: x\*\*2 - 3\*x + 2 = 0
2. Solve by factoring: x\*\*2 - 5\*x + 6 = 0
3. Solve by factoring: x\*\*2 - 7\*x + 12 = 0

## Solution Key

1. Answer: 1, 2

   - Start with x\*\*2 - 3\*x + 2 = 0.
   - Factor the quadratic as (x - 1)(x - 2) = 0.
   - Set each factor equal to zero.
   - x = 1 or x = 2.
2. Answer: 2, 3

   - Start with x\*\*2 - 5\*x + 6 = 0.
   - Factor the quadratic as (x - 2)(x - 3) = 0.
   - Set each factor equal to zero.
   - x = 2 or x = 3.
3. Answer: 3, 4

   - Start with x\*\*2 - 7\*x + 12 = 0.
   - Factor the quadratic as (x - 3)(x - 4) = 0.
   - Set each factor equal to zero.
   - x = 3 or x = 4.
