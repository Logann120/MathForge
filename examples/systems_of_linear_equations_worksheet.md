# Systems of linear equations Worksheet

## Instructions

Solve each system of equations for x and y.

## Problems

1. Solve the system of equations:
x + y = 3
x - y = -1
2. Solve the system of equations:
x + y = 5
x - y = -1
3. Solve the system of equations:
x + y = 7
x - y = -1

## Solution Key

1. Answer: (1, 2)

   - Start with the system x + y = 3 and x - y = -1.
   - Add the equations to eliminate y.
   - Solve for x to get x = 1.
   - Substitute x = 1 into one equation to get y = 2.
   - The solution is (1, 2).
2. Answer: (2, 3)

   - Start with the system x + y = 5 and x - y = -1.
   - Add the equations to eliminate y.
   - Solve for x to get x = 2.
   - Substitute x = 2 into one equation to get y = 3.
   - The solution is (2, 3).
3. Answer: (3, 4)

   - Start with the system x + y = 7 and x - y = -1.
   - Add the equations to eliminate y.
   - Solve for x to get x = 3.
   - Substitute x = 3 into one equation to get y = 4.
   - The solution is (3, 4).
