# Functions basics Worksheet

## Instructions

Evaluate and interpret each function notation problem.

## Problems

1. Given f(x) = 2\*x + 1, evaluate f(3).
2. In the statement f(4) = 9, what does 4 represent?
3. Determine the domain of f(x) = 1/(x - 2).

## Solution Key

1. Answer: 7

   - Start with f(x) = 2\*x + 1.
   - Substitute x = 3.
   - Evaluate to get f(3) = 7.
2. Answer: The input value

   - Read f(4) as the output of f when x = 4.
   - The value 4 is the input.
   - The value 9 is the output.
3. Answer: All real numbers except x = 2

   - Start with f(x) = 1/(x - 2).
   - The denominator cannot equal zero.
   - Exclude x = 2 from the domain.
   - The domain is All real numbers except x = 2.
