Systems of linear equations Worksheet

Instructions

Solve each system of equations for x and y.

Problems

  1. Solve the system of equations: x + y = 3 x - y = -1
  2. Solve the system of equations: x + y = 5 x - y = -1
  3. Solve the system of equations: x + y = 7 x - y = -1

Solution Key

  1. Answer: (1, 2)

    1. Start with the system x + y = 3 and x - y = -1.
    2. Add the equations to eliminate y.
    3. Solve for x to get x = 1.
    4. Substitute x = 1 into one equation to get y = 2.
    5. The solution is (1, 2).

  2. Answer: (2, 3)

    2. Add the equations to eliminate y.
    3. Solve for x to get x = 2.
    4. Substitute x = 2 into one equation to get y = 3.
    5. The solution is (2, 3).
