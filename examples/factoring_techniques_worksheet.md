# Factoring techniques Worksheet

## Instructions

Factor each polynomial expression completely.

## Problems

1. Factor completely: 6\*x + 8
2. Factor completely: x\*\*2 - 4
3. Factor completely: x\*\*2 + 5\*x + 6

## Solution Key

1. Answer: 2\*(3\*x + 4)

   - Start with 6\*x + 8.
   - Identify the strategy: greatest common factor.
   - Rewrite the expression as 2\*(3\*x + 4).
   - Check by expanding the factored form.
2. Answer: (x - 2)\*(x + 2)

   - Start with x\*\*2 - 4.
   - Identify the strategy: difference of squares.
   - Rewrite the expression as (x - 2)\*(x + 2).
   - Check by expanding the factored form.
3. Answer: (x + 2)\*(x + 3)

   - Start with x\*\*2 + 5\*x + 6.
   - Identify the strategy: simple trinomial.
   - Rewrite the expression as (x + 2)\*(x + 3).
   - Check by expanding the factored form.
