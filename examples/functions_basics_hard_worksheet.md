# Functions basics Worksheet

## Instructions

Evaluate and interpret each function notation problem.

## Problems

1. Given f(x) = 2\*x + 1 and g(x) = x\*\*2, evaluate f(g(2)).
2. Determine the domain of f(x) = 1/((x - 2)\*(x + 3)).
3. Given f(x) = 3\*x + 2 and g(x) = x\*\*2, evaluate f(g(3)).

## Solution Key

1. Answer: 9

   - Start by evaluating g(2) using g(x) = x\*\*2.
   - g(2) = 4.
   - Now evaluate f(4) using f(x) = 2\*x + 1.
   - f(g(2)) = 9.
2. Answer: All real numbers except x = -3 and x = 2

   - Start with f(x) = 1/((x - 2)\*(x + 3)).
   - The denominator cannot equal zero.
   - Set each factor equal to zero to find x = 2 and x = -3.
   - The domain is All real numbers except x = -3 and x = 2.
3. Answer: 29

   - Start by evaluating g(3) using g(x) = x\*\*2.
   - g(3) = 9.
   - Now evaluate f(9) using f(x) = 3\*x + 2.
   - f(g(3)) = 29.
