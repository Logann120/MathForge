# Factoring techniques Worksheet

## Instructions

Factor each polynomial expression completely.

## Problems

1. Factor completely: x\*\*3 + 3\*x\*\*2 + 2\*x + 6
2. Factor completely: 2\*x\*\*2 + 7\*x + 3
3. Factor completely: x\*\*3 + 4\*x\*\*2 + 3\*x + 12

## Solution Key

1. Answer: (x + 3)\*(x\*\*2 + 2)

   - Start with x\*\*3 + 3\*x\*\*2 + 2\*x + 6.
   - Identify the strategy: factor by grouping.
   - Rewrite the expression as (x + 3)\*(x\*\*2 + 2).
   - Check by expanding the factored form.
2. Answer: (2\*x + 1)\*(x + 3)

   - Start with 2\*x\*\*2 + 7\*x + 3.
   - Identify the strategy: non-monic trinomial.
   - Rewrite the expression as (2\*x + 1)\*(x + 3).
   - Check by expanding the factored form.
3. Answer: (x + 4)\*(x\*\*2 + 3)

   - Start with x\*\*3 + 4\*x\*\*2 + 3\*x + 12.
   - Identify the strategy: factor by grouping.
   - Rewrite the expression as (x + 4)\*(x\*\*2 + 3).
   - Check by expanding the factored form.
