# Systems of linear equations Worksheet

## Instructions

Solve each system of equations for x and y.

## Problems

1. Solve the system of equations:
-2\*x + 3\*y = -7
4\*x + y = 7
2. Solve the system of equations:
-2\*x + 3\*y = -12
4\*x + y = 10
3. Solve the system of equations:
-2\*x + 3\*y = -17
4\*x + y = 13

## Solution Key

1. Answer: (2, -1)

   - Start with the system -2\*x + 3\*y = -7 and 4\*x + y = 7.
   - Multiply the second equation by -3 so the y-coefficients are opposites.
   - Add the equations to eliminate y.
   - Solve for x to get x = 2.
   - Substitute x = 2 into one equation to get y = -1.
   - The solution is (2, -1).
2. Answer: (3, -2)

   - Start with the system -2\*x + 3\*y = -12 and 4\*x + y = 10.
   - Multiply the second equation by -3 so the y-coefficients are opposites.
   - Add the equations to eliminate y.
   - Solve for x to get x = 3.
   - Substitute x = 3 into one equation to get y = -2.
   - The solution is (3, -2).
3. Answer: (4, -3)

   - Start with the system -2\*x + 3\*y = -17 and 4\*x + y = 13.
   - Multiply the second equation by -3 so the y-coefficients are opposites.
   - Add the equations to eliminate y.
   - Solve for x to get x = 4.
   - Substitute x = 4 into one equation to get y = -3.
   - The solution is (4, -3).
