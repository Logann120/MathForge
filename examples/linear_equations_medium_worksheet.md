# Linear equations Worksheet

## Instructions

Solve each equation for x.

## Problems

1. Solve for x: 5\*x - 3 = -8
2. Solve for x: 6\*x - 5 = -17
3. Solve for x: 7\*x - 7 = -28

## Solution Key

1. Answer: -1

   - Start with 5\*x - 3 = -8.
   - Add 3 to both sides.
   - Divide both sides by 5.
   - x = -1.
2. Answer: -2

   - Start with 6\*x - 5 = -17.
   - Add 5 to both sides.
   - Divide both sides by 6.
   - x = -2.
3. Answer: -3

   - Start with 7\*x - 7 = -28.
   - Add 7 to both sides.
   - Divide both sides by 7.
   - x = -3.
